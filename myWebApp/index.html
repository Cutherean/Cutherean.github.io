About the Project

This web map is a choropleth map used to show Internet usage over time.

The data used in this map is modified from the original source. Many countries such as Taiwan and Antarctica were not present in the map and would not load properly with the topoJSON file. Additionally, the data would show 0 for years with no data instead of an alternate placeholder. To remedy both of these, a Python script was used.
